API Endpoints


# Quiz Management
GET     /api/quizzes              - List all quizzes
POST    /api/quizzes              - Create new quiz
GET     /api/quizzes/<id>       - Get quiz by ID
PUT     /api/quizzes/<id>       - Update quiz
DELETE  /api/quizzes/<id>       - Delete quiz

# Answer Questions
POST    /api/answer/<id>        - Submit answer

# AI Generator
POST    /api/generator/generate  - Generate & save quizzes
POST    /api/generator/preview   - Preview generated quizzes

# Pages
GET     /                         - Home page
GET     /quiz/<id>             - Quiz detail page
GET     /generator               - Generator page
