# Docs — Thiết kế dự án Quiz_En

Tập hợp các tài liệu thiết kế của dự án: kiến trúc, route, service, deployment.

Các file:
- `architecture.md` — Tổng quan kiến trúc và sơ đồ luồng dữ liệu
- `routes.md` — Quy ước và danh sách endpoint hiện tại
- `services.md` — Thiết kế lớp business logic và hướng dẫn viết tests
- `deployment.md` — Hướng dẫn chạy và triển khai (prod tips)

---

Mở file tương ứng để xem chi tiết.
